============
Installation
============


Stable release
--------------

To install Ethiopian Date Converter, run this command in your terminal:

.. code-block:: console

    $ pip install ethiopian_date_converter

This is the preferred method to install Ethiopian Date Converter, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Ethiopian Date Converter can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/jupi2142/ethiopian_date_converter

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/jupi2142/ethiopian_date_converter/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/jupi2142/ethiopian_date_converter
.. _tarball: https://github.com/jupi2142/ethiopian_date_converter/tarball/master
